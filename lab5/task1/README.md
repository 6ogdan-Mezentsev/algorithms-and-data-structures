## Задание
Структуру данных «куча», или, более конкретно, «неубывающая пирамида», можно реализовать на основе массива.
Дан массив целых чисел. Определите, является ли он неубывающей пирамидой.

## Input / Output 

| Input     | Output |
|-----------|--------|
| 5         | NO     |
| 1 0 1 2 0 |
