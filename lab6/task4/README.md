## Задание
Вам дается последовательность чисел. Для каждого числа определите, явля-ется ли оно числом Фибоначчи. Напомним, что числа Фибоначчи определяются, например, так:

Fo = F1=1

Fi = Fi-1 + Fi-2 для і ≥ 2.

## Input / Output 

| Input | Output |
|-------|--------|
| 8     | Yes    | 
| 1     | Yes    |
| 2     | Yes    |
| 3     | No     |
| 4     | Yes    |
| 5     | No     |
| 6     | No     |
| 7     | Yes    |
| 8     |        |
